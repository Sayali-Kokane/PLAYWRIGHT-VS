# PLAYWRIGHT-VS

Minimal Playwright / Node learning repository.

## Contents

- `chapt1_js_basics/01_basics.js` — prints "Hello World!"
- `chapt1_js_basics/02_js step_by_step` — prints `10`

## Requirements

- Node.js (v14+). Tested with v24.

## Run

From repository root run:

```sh
node "chapt1_js_basics/01_basics.js"
```

or

```sh
node "chapt1_js_basics/02_js step_by_step"
```
